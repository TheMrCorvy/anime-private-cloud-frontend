### Running the project locally

```bash
npm run dev
```

### Execute the tests

```bash
npm run test
```

##### Execute the tests (with coverage)

```bash
npm run test:coverage
```

### Execute the linter

```bash
npm run lint
```

### Run Storybook

```bash
npm run storybook
```
